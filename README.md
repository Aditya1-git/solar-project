# ☀️ Solar System

An interactive Solar System project that visualizes the planets orbiting around the Sun. The project focuses on creating an engaging and educational experience while demonstrating modern web development concepts such as animations, responsive layouts, and interactive UI.

## 🚀 Features

- 🌍 Interactive Solar System visualization
- 🪐 Animated planetary orbits
- ✨ Smooth CSS animations
- 📱 Responsive design
- 🎨 Clean and modern user interface
- ⚡ Lightweight and fast

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
